About My Site

Building the website by myself helps me form a more comprehensive understanding about website design. I realized designing the webpage is quite different from doing graphic design using illustrator or InDesign. While using html and CSS, every element needs to be given a precise value such as the width, height, margin, and padding. Moreover, designing a webpage also need to consider the final effect in different sizes of screens, which means I should always work with proportions rather than pixels.


Furthermore, there are something I would done different if I realized earlier. As for the CSS, I used to create a different CSS file for different components in a webpage such as the navbar, the main body, the image part. However, I found there will be conflicts if two CSS files mention the same component. This brought me a lot of troubles to figure out which file should I change. I will minimize the CSS file and use class and id to identify different elements in the future. As for the gallery part, if I had realized earlier, I would use images with lower resolution as a thumbnail image rather than the original image to make the page loading faster.
